# Plex LinuxServer

## 产品介绍

Plex LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/plex` 镜像，提供 Plex 媒体服务器 能力。

## 主要功能

- 提供 媒体库管理与串流播放 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 Plex Web UI；首次绑定服务器时可填写 Plex Claim Token。

## Introduction

Plex LinuxServer uses the LinuxServer.io maintained `linuxserver/plex` image for media library management and streaming.

## Features

- Provide media library management and streaming
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Plex Web UI from the HTTP port; provide a Plex Claim Token when claiming a new server.
