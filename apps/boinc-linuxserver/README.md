# BOINC LinuxServer

## 产品介绍

BOINC LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/boinc` 镜像，提供 BOINC 分布式计算桌面 能力。

## 主要功能

- 提供 BOINC 分布式计算管理 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 HTTP 端口访问 BOINC Web 桌面。

## Introduction

BOINC LinuxServer uses the LinuxServer.io maintained `linuxserver/boinc` image for BOINC distributed computing management.

## Features

- Provide BOINC distributed computing management
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, open the BOINC Web desktop from the 1Panel app entry or the HTTP port.
