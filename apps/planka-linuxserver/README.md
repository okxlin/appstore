# Planka
## 产品介绍

Planka 使用 LinuxServer.io 维护的 `linuxserver/planka` 镜像，提供 Planka 看板项目管理 能力。

## 主要功能

- 提供 看板项目管理 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 Planka，并使用表单中配置的管理员账号登录。

## Introduction

Planka uses the LinuxServer.io maintained `linuxserver/planka` image for Kanban project management.

## Features

- Provide Kanban project management
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open Planka from the HTTP port and sign in with the configured admin account.
