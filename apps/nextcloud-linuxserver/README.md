# Nextcloud LinuxServer

## 产品介绍

Nextcloud LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/nextcloud` 镜像，提供 Nextcloud 私有云协作平台 能力。

## 主要功能

- 提供 私有云文件同步与协作 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTPS 端口访问 Nextcloud Web 界面并完成初始化向导。

## Introduction

Nextcloud LinuxServer uses the LinuxServer.io maintained `linuxserver/nextcloud` image for private cloud file sync and collaboration.

## Features

- Provide private cloud file sync and collaboration
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Nextcloud web interface from the HTTPS port and complete the setup wizard.
