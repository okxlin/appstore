# Ting Reader

Ting Reader 是一个轻量级的自托管有声书平台，专为有声书爱好者打造。

## 功能特性

- ⚡ Rust 核心，资源占用极低，响应速度极快
- 📚 自动刮削元数据，支持喜马拉雅等平台
- 🔌 插件系统，支持 JS、WASM 和 Native 插件
- 🎨 自适应主题，根据封面自动调色
- 🎵 支持 MP3, M4A, M4B, WAV, FLAC, OGG, OPUS, AAC, WMA等主流格式
- 🎧 沉浸式播放，支持跳过片头/片尾、速度调节、进度记忆
- 🌓 完美的深色模式
- 🔐 完善的权限管理

## 默认账号

- 用户名：`admin`
- 密码：`admin123`

⚠️ 首次登录后请务必修改默认密码！

## 使用说明

1. 安装完成后访问应用
2. 使用默认账号登录
3. 在管理后台添加存储库，指向有声书文件目录
4. 系统会自动扫描并刮削元数据
5. 开始享受有声书之旅！
