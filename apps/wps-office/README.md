# WPS Office

## 产品介绍

WPS Office 是一套轻量级办公套件，支持文字、表格、演示和 PDF 等常见办公文档处理。LinuxServer 版本通过浏览器提供桌面界面。

## 主要功能

- 在浏览器中使用 WPS Office 桌面环境
- 支持 Writer、Presentation、Spreadsheet 和 PDF 工作流
- 提供 HTTP/HTTPS 访问端口
- 支持配置目录持久化，便于保留用户设置和文档

## 访问说明

安装完成后，通过应用配置的 HTTP 或 HTTPS 端口访问 WPS Office 桌面界面。

## Introduction

WPS Office is a lightweight office suite for documents, spreadsheets, presentations, and PDF workflows. The LinuxServer version exposes the desktop interface through a browser.

## Features

- Use the WPS Office desktop environment from a browser
- Work with Writer, Presentation, Spreadsheet, and PDF documents
- Exposes HTTP and HTTPS access ports
- Supports persistent config storage for user settings and documents

## Access

After installation, open WPS Office from the configured HTTP or HTTPS port.
