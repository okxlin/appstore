# DeepLX

## 应用简介
DeepL 免费 API（无需 TOKEN）。

英文说明：DeepL Free API (No TOKEN required).

## 部署说明
- 本应用使用 Docker Compose 在 1Panel 中部署。
- 应用分类：工具。
- 支持架构：amd64。
- 可选版本：`latest`、`1.2.2`。
- 安装后按应用表单中的端口访问 Web UI、SSH 或对应服务。

## 端口
| 变量 | 说明 | 默认值 | 必填 |
| --- | --- | --- | --- |
| PANEL_APP_PORT_HTTP | 端口 | 40174 | 是 |

## 配置项
| 变量 | 说明 | 默认值 | 必填 |
| --- | --- | --- | --- |
| TOKEN_VAULE | 密码 | deeplx | 否 |

## 使用说明
使用例子`http://IP:Port/translate`
