# FreeTube LinuxServer

## 产品介绍

FreeTube LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/freetube` 镜像，提供 FreeTube 视频客户端桌面 能力。

## 主要功能

- 提供 FreeTube 桌面访问 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 HTTP 端口访问 FreeTube 桌面。

## Introduction

FreeTube LinuxServer uses the LinuxServer.io maintained `linuxserver/freetube` image for FreeTube desktop access.

## Features

- Provide FreeTube desktop access
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, open the FreeTube desktop from the 1Panel app entry or the HTTP port.
