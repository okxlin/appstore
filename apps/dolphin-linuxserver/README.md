# Dolphin LinuxServer

## 产品介绍

Dolphin LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/dolphin` 镜像，提供 Dolphin 游戏模拟器桌面 能力。

## 主要功能

- 提供 游戏模拟器桌面 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 或 HTTPS 端口访问 Dolphin Web 桌面。

## Introduction

Dolphin LinuxServer uses the LinuxServer.io maintained `linuxserver/dolphin` image for game emulator desktop.

## Features

- Provide game emulator desktop
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Dolphin web desktop from the HTTP or HTTPS port.
