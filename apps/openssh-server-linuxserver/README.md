# OpenSSH Server LinuxServer

## 产品介绍

OpenSSH Server LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/openssh-server` 镜像，提供 OpenSSH SSH 服务 能力。

## 主要功能

- 提供 SSH 服务 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，使用 SSH 客户端连接服务器地址和 SSH 端口；可在配置目录中维护 authorized_keys。

## Introduction

OpenSSH Server LinuxServer uses the LinuxServer.io maintained `linuxserver/openssh-server` image for SSH service.

## Features

- Provide SSH service
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, connect with an SSH client using the server address and SSH port; authorized_keys can be maintained in the config directory.
