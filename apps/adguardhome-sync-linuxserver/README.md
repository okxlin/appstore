# AdGuardHome Sync LinuxServer

## 产品介绍

AdGuardHome Sync LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/adguardhome-sync` 镜像，提供 AdGuardHome Sync 配置同步服务 能力。

## 主要功能

- 提供 AdGuard Home 配置同步 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 AdGuardHome Sync Web API，并在配置目录中维护同步配置。

## Introduction

AdGuardHome Sync LinuxServer uses the LinuxServer.io maintained `linuxserver/adguardhome-sync` image for AdGuard Home configuration synchronization.

## Features

- Provide AdGuard Home configuration synchronization
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, use the HTTP port for the AdGuardHome Sync web API and maintain sync configuration in the config directory.
