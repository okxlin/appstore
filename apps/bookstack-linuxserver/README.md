# BookStack LinuxServer

## 产品介绍

BookStack LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/bookstack` 镜像，提供 BookStack 文档知识库 能力。

## 主要功能

- 提供 文档知识库 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 BookStack，默认账号为 admin@admin.com / password。

## Introduction

BookStack LinuxServer uses the LinuxServer.io maintained `linuxserver/bookstack` image for documentation wiki.

## Features

- Provide documentation wiki
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open BookStack from the HTTP port. The default account is admin@admin.com / password.
