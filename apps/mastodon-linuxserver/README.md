# Mastodon
## 产品介绍

Mastodon 使用 LinuxServer.io 维护的 `linuxserver/mastodon` 镜像，提供 Mastodon ActivityPub 社交网络服务 能力。

## 主要功能

- 提供 去中心化社交网络服务 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 或 HTTPS 端口访问 Mastodon；生产使用前请替换安装表单中的域名和密钥。

## Introduction

Mastodon uses the LinuxServer.io maintained `linuxserver/mastodon` image for federated social networking.

## Features

- Provide federated social networking
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open Mastodon from the HTTP or HTTPS port; replace the domain and secret fields before production use.
