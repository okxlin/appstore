# HABridge
## 产品介绍

HABridge 使用 LinuxServer.io 维护的 `linuxserver/habridge` 镜像，提供 HABridge 智能家居桥接 能力。

## 主要功能

- 提供 智能家居桥接 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 HTTP 端口访问 HABridge Web 界面；如需设备发现可按需开放通信端口。

## Introduction

HABridge uses the LinuxServer.io maintained `linuxserver/habridge` image for home automation bridging.

## Features

- Provide home automation bridging
- Persist configuration and application data
- Configure service ports from the install form when the image exposes ports
- Configure the container time zone

## Access

After installation, open the HABridge web interface from the 1Panel app entry or the HTTP port; expose the communication port when device discovery needs it.
