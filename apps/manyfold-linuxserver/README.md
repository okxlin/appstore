# Manyfold
## 产品介绍

Manyfold 使用 LinuxServer.io 维护的 `linuxserver/manyfold` 镜像，提供 Manyfold 3D 模型管理 能力。

## 主要功能

- 提供 3D 模型库管理 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 Manyfold Web 界面。默认使用 SQLite 数据库。

## Introduction

Manyfold uses the LinuxServer.io maintained `linuxserver/manyfold` image for 3D model library management.

## Features

- Provide 3D model library management
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Manyfold web interface from the HTTP port. The default database is SQLite.
