# Chromium
## 产品介绍

Chromium 使用 LinuxServer.io 维护的 `linuxserver/chromium` 镜像，提供 Chromium 浏览器桌面 能力。

## 主要功能

- 提供 浏览器桌面 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 或 HTTPS 端口访问 Chromium Web 桌面。

## Introduction

Chromium uses the LinuxServer.io maintained `linuxserver/chromium` image for browser desktop.

## Features

- Provide browser desktop
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Chromium web desktop from the HTTP or HTTPS port.
