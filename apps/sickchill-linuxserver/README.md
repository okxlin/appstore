# SickChill
## 产品介绍

SickChill 使用 LinuxServer.io 维护的 `linuxserver/sickchill` 镜像，提供 SickChill 电视节目管理 能力。

## 主要功能

- 提供 电视节目管理 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 HTTP 端口访问 SickChill Web 界面。

## Introduction

SickChill uses the LinuxServer.io maintained `linuxserver/sickchill` image for TV show management.

## Features

- Provide TV show management
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, open the SickChill web interface from the 1Panel app entry or the HTTP port.
