# WebGrabPlus LinuxServer

## 产品介绍

WebGrabPlus LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/webgrabplus` 镜像，提供 WebGrabPlus 电视节目抓取服务 能力。

## 主要功能

- 提供 电视节目表抓取 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，在配置目录中维护 WebGrabPlus 配置，输出文件保存在数据目录；该应用没有 Web 界面。

## Introduction

WebGrabPlus LinuxServer uses the LinuxServer.io maintained `linuxserver/webgrabplus` image for TV guide grabbing.

## Features

- Provide TV guide grabbing
- Persist configuration and application data
- Configure service ports from the install form when the image exposes ports
- Configure the container time zone

## Access

After installation, maintain WebGrabPlus configuration in the config directory and read generated output from the data directory; this app does not provide a web interface.
