# ngIRCd
## 产品介绍

ngIRCd 使用 LinuxServer.io 维护的 `linuxserver/ngircd` 镜像，提供 ngIRCd IRC 服务 能力。

## 主要功能

- 提供 IRC 服务 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，在 IRC 客户端中填写服务器地址和 IRC 端口连接服务。

## Introduction

ngIRCd uses the LinuxServer.io maintained `linuxserver/ngircd` image for IRC service.

## Features

- Provide IRC service
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, connect with an IRC client using the server address and IRC port.
