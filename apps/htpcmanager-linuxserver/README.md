# HTPC Manager LinuxServer

## 产品介绍

HTPC Manager LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/htpcmanager` 镜像，提供 HTPC Manager 家庭媒体控制台 能力。

## 主要功能

- 提供 家庭媒体服务控制台 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 HTPC Manager Web 界面。

## Introduction

HTPC Manager LinuxServer uses the LinuxServer.io maintained `linuxserver/htpcmanager` image for HTPC service dashboard.

## Features

- Provide HTPC service dashboard
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the HTPC Manager web interface from the HTTP port.
