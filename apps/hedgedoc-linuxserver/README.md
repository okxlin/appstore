# HedgeDoc LinuxServer

## 产品介绍

HedgeDoc LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/hedgedoc` 镜像，提供 HedgeDoc 协作文档编辑器 能力。

## 主要功能

- 提供 协作 Markdown 文档编辑 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 HedgeDoc。

## Introduction

HedgeDoc LinuxServer uses the LinuxServer.io maintained `linuxserver/hedgedoc` image for collaborative markdown editing.

## Features

- Provide collaborative markdown editing
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open HedgeDoc from the HTTP port.
