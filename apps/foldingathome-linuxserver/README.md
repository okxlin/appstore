# Folding@home
## 产品介绍

Folding@home 使用 LinuxServer.io 维护的 `linuxserver/foldingathome` 镜像，提供 Folding@home 分布式计算客户端 能力。

## 主要功能

- 提供 分布式计算任务运行 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 端口访问 Folding@home Web Control。

## Introduction

Folding@home uses the LinuxServer.io maintained `linuxserver/foldingathome` image for distributed computing work.

## Features

- Provide distributed computing work
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open Folding@home Web Control from the HTTP port.
