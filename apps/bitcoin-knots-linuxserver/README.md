# Bitcoin Knots LinuxServer

## 产品介绍

Bitcoin Knots LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/bitcoin-knots` 镜像，提供 Bitcoin Knots 全节点桌面 能力。

## 主要功能

- 提供 Bitcoin 全节点管理 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 HTTP 端口访问 Bitcoin Knots 桌面；如需入站节点连接可开放 P2P 端口。

## Introduction

Bitcoin Knots LinuxServer uses the LinuxServer.io maintained `linuxserver/bitcoin-knots` image for Bitcoin full node management.

## Features

- Provide Bitcoin full node management
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, open the Bitcoin Knots desktop from the 1Panel app entry or the HTTP port; expose the P2P port if inbound node connectivity is needed.
