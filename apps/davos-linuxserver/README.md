# Davos LinuxServer

## 产品介绍

Davos LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/davos` 镜像，提供 Davos 文件下载管理 能力。

## 主要功能

- 通过 Web UI 进行 download management 配置
- 持久化保存配置和业务数据
- 使用 8080 端口提供 Web 管理界面
- 支持自定义时区

## 访问说明

安装完成后，通过 1Panel 应用入口或 `http://服务器地址:HTTP端口` 访问 Web 界面。首次启动后请按应用页面提示完成初始化配置。

## Introduction

Davos LinuxServer uses the LinuxServer.io maintained `linuxserver/davos` image for download management.

## Features

- Manage the application from the Web UI
- Persist configuration and application data
- Expose the Web interface on port 8080
- Configure the container time zone
