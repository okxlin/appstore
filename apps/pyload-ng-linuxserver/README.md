# pyLoad NG LinuxServer

## 产品介绍

pyLoad NG 是一个轻量下载管理器，提供 Web 管理界面和 ClickNLoad 支持。本应用使用 LinuxServer.io 维护的 `linuxserver/pyload-ng` 镜像。

## 主要功能

- 通过 Web UI 管理下载任务和插件
- 持久化保存配置和下载目录
- 使用 8000 端口提供 Web 管理界面
- 可选暴露 ClickNLoad 9666 端口

## 访问说明

安装完成后，通过 1Panel 应用入口或 `http://服务器地址:HTTP端口` 访问 Web 界面。首次启动后请按应用页面提示完成初始化、账号或后端服务配置。

## Introduction

pyLoad NG is a lightweight download manager with a Web interface and ClickNLoad support. This app uses the LinuxServer.io maintained `linuxserver/pyload-ng` image.

## Features

- Manage downloads and plugins from the Web UI
- Persist configuration and downloads
- Expose the Web interface on port 8000
- Optionally expose the ClickNLoad port 9666
