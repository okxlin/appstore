# Minetest LinuxServer

## 产品介绍

Minetest LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/minetest` 镜像，提供 Minetest 沙盒游戏服务器 能力。

## 主要功能

- 提供 Minetest 游戏服务器 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，使用 Minetest 客户端连接服务器地址和 UDP 游戏端口。

## Introduction

Minetest LinuxServer uses the LinuxServer.io maintained `linuxserver/minetest` image for Minetest game server.

## Features

- Provide Minetest game server
- Persist configuration and application data
- Configure service ports from the install form
- Configure the container time zone

## Access

After installation, connect with a Minetest client using the server address and UDP game port.
