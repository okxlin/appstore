# ddclient LinuxServer

## 产品介绍

ddclient LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/ddclient` 镜像，提供 ddclient 动态 DNS 更新服务 能力。

## 主要功能

- 提供 动态 DNS 更新 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口
- 支持自定义时区

## 访问说明

安装完成后，在配置目录中维护 ddclient.conf；该应用没有 Web 界面。

## Introduction

ddclient LinuxServer uses the LinuxServer.io maintained `linuxserver/ddclient` image for dynamic DNS updating.

## Features

- Provide dynamic DNS updating
- Persist configuration and application data
- Configure service ports from the install form when the image exposes ports
- Configure the container time zone

## Access

After installation, maintain ddclient.conf in the config directory; this app does not provide a web interface.
