# Kimai LinuxServer

## 产品介绍

Kimai LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/kimai` 镜像，提供 Kimai 时间跟踪系统 能力。

## 主要功能

- 提供 时间跟踪与工时统计 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 或 HTTPS 端口访问 Kimai，并在 Web 界面中完成初始化。

## Introduction

Kimai LinuxServer uses the LinuxServer.io maintained `linuxserver/kimai` image for time tracking and timesheet reporting.

## Features

- Provide time tracking and timesheet reporting
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open Kimai from the HTTP or HTTPS port and complete setup in the web interface.
