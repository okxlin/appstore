# Weixin LinuxServer

## 产品介绍

Weixin LinuxServer 使用 LinuxServer.io 维护的 `linuxserver/weixin` 镜像，提供 微信桌面客户端 能力。

## 主要功能

- 提供 微信桌面客户端 能力
- 持久化保存配置和业务数据
- 使用安装表单配置服务端口和数据路径
- 支持自定义时区

## 访问说明

安装完成后，通过 HTTP 或 HTTPS 端口访问微信 Web 桌面。

## Introduction

Weixin LinuxServer uses the LinuxServer.io maintained `linuxserver/weixin` image for Weixin desktop client.

## Features

- Provide Weixin desktop client
- Persist configuration and application data
- Configure service ports and data paths from the install form
- Configure the container time zone

## Access

After installation, open the Weixin web desktop from the HTTP or HTTPS port.
